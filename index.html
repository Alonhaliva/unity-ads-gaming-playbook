LEVEL 01 · ROAS CAMPAIGNS

THE MAIN
QUEST.

ROAS optimization is the core loop of the gaming vertical. Vector AI bids to return on ad spend across IAP, ad revenue, and hybrid — pick the window that matches the client's monetization.

D7+D28 RUN BOTH WINDOWS — BEST PRACTICE
>75 D7 PURCHASERS TO EXIT LEARNING (IAP)
NO CPI WARM-UP REQUIRED ANYMORE
Campaign types Learning phase How to pitch

IAP D7 / D28 ROAS — optimizes return from in-app purchases over the 7- or 28-day window. D7 catches fast monetizers; D28 matches higher long-term value. Run both concurrently for best game-level performance.

Ad Revenue D0 / D7 ROAS — bids to ad-revenue returns from acquired players (D28 in alpha). Built for IAA / hybrid-casual titles.

Hybrid D7 ROAS (live; D28 alpha) — optimizes IAP + ad revenue simultaneously. The default for modern hybrid titles where both revenue streams matter.

Learning is now measured at the Game ID level across all targeted geos — not per country. Once a game banks enough total data, every ROAS campaign of that type flips to Live at once. 90-day volume thresholds:

  • IAP D7: >75 unique D7 purchasers
  • Ad Revenue D0 / D7: >1,000 installs and >$0 ad revenue
  • Hybrid D7: >1,000 installs, >$0 ad revenue (D7), >35 unique D7 purchasers

Lead with: "You can launch ROAS from day one — no CPI warm-up burning budget, and learning completes at the game level so all geos go live together." Then match the window to their model: pure IAP → D7+D28 IAP; ad-monetized hyper/hybrid-casual → Ad Revenue D0/D7; modern hybrid → Hybrid D7. Confirm MMP is clean first (see Level 03).

LEVEL 03 · VECTOR & PROJECT FORGE

THE GAME
ENGINE.

Everything now runs on one engine. In early 2026 Project Forge sunset the ironSource Ads direct demand network — demand is fully consolidated on Unity Ads powered by Vector AI, LevelPlay mediation, and Tapjoy.

What Vector AI does

Vector is the machine-learning matching engine at the core of every UA campaign. It reads player behavioral signals and matches users across IAP, IAA, and hybrid models — the same brain optimizing ROAS bids, creative cold-start, and multi-country budgets. When you sell "Unity Ads," you're selling Vector.

Project Forge — what changed
ironSource Ads direct demand network sunset. One demand stack: Unity Ads + Vector. One mediation: LevelPlay. One offerwall: Tapjoy. Cleaner story for clients, no more "which network?" confusion — but expect questions from ex-ironSource accounts about migration.
⚠ MMP VALIDATION IS NOW ENFORCED
The ROAS onboarding flow auto-checks MMP config. P0 (critical) errors — mismatched Game IDs, postback windows under 180 days, wrong event mappings — block the campaign from going live. Since April 9, 2026, AppsFlyer clients failing P0 validation cannot launch until they fix setup. Always validate MMP before promising a launch date. P1 rules (share unattributed + organic data) are recommended, not blocking.

LEVEL 04 · BOLD SDK

POWER-UP
PATCH.

"Bold" is Unity's rebuilt ad-delivery framework, designed from the ground up for mediation and header bidding. The upgrade is a free performance power-up — push every client to it.

5x FASTER INIT · P95 2.37s · ~46% FASTER
~91% LOAD ERROR RATE DROP → 1%
~35% SHOW ERROR RATE DROP → 2%

Upgrade cheat-sheet

  • Recommend SDK 4.17.0+ — preferably 4.17.1
  • iOS / StoreKit 2: pair latest Ads SDK with IAP Plugin 5.0.4 for clean purchase signals feeding Vector
  • Android low-memory: 4.16.6+ ships the OOM crash stability fix — upgrade any client reporting low-mem crashes
  • OM SDK support enables viewability metrics → qualifies inventory for premium brand demand

Clean purchase signals are not cosmetic — they directly feed Vector's optimization. A client on an old SDK with a broken StoreKit pairing is starving the algorithm. SDK upgrade = better ROAS. That's the one-line pitch.

LEVEL 07 · BOSS FIGHTS

KNOW YOUR
ENEMY.

The networks coming for our gaming clients. Know each one's move set before the call.

▸ AppLovin (MAX) — the final boss
Default mediation for many studios. Feedback is mixed: clients gripe about weak support and postback issues (Atlas flagged this), but inertia is real — "happy on MAX, not looking to switch." Counter with LevelPlay support + Vector performance; don't attack creatives (clients run the same ones everywhere).
▸ Mintegral · Moloco · Google · Meta
The standing UA roster on most hybrid/midcore accounts (per Voodoo, MagicLab). They hold share where Unity sits under 10%. Same creatives, same profitability — so win on ROAS proof, D28 windows, and SDK performance, not price.
▸ Liftoff — newly public ($3.83B)
Liftoff IPO'd this week at $3.83B — expect more aggressive UA spend and a sharpened sales motion funded by public-market pressure. Watch performance-network accounts; they'll lean into machine-learning bidding parity.
▸ TikTok — fast-rising channel
Now a top-3 channel for many advertisers, sometimes #2. Clients push rewarded + social traffic and increasingly want TikTok's SDK. Not a like-for-like network competitor, but it's taking budget share — frame Unity as the performance/ROAS engine TikTok isn't.
▸ Adjoe — offerwall threat
140% YoY regional growth, just appointed a LATAM lead (June 9) and posted 1000% YoY in South Korea. Direct overlap with Tapjoy Offerwall in incentive/rewarded. If a client runs incentive traffic, assume Adjoe is pitching them.

How to win the fight

  • Lead with Vector + D28 ROAS proof — it's the clearest performance story
  • On "competitor is cheaper / better ROAS" — anchor on the same creatives + clean MMP, not discounts
  • For sub-10% accounts (MagicLab), the unlock is share growth via SDK upgrade + Hybrid ROAS
  • Don't bad-mouth AppLovin — acknowledge support gaps and out-execute on attention
  • For incentive/rewarded clients, position Tapjoy directly against Adjoe
